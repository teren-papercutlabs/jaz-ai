### Import Chart Of Accounts (COA)
Source: https://help.jaz.ai/en/articles/9115783-import-chart-of-accounts-coa

**Q1. Can I import accounts in bulk?**

- Yes, you can import accounts in bulk using our **Import Accounts**template.
- From **Settings >Configurations > Chart of Accounts > Import Accounts**you can import your accounts using Jaz's template
- Follow the steps on-screen to start importing accounts.
- Review the imported list of accounts. You should see accounts that will be added or updated.
- If there are no issues, finish importing your accounts.
  - If there are errors, continue adjusting and modifying your template, and click on **Back**to reupload.

**Q2. Can I update the existing COA using import COA?**

- Yes, you can. When downloading the **Import Accounts** template, the template will be pre-filled with existing accounts that you currently already have on Jaz.
- You can modify the information in these rows relating to existing accounts.
  - For example, you may want to change the name of one of your existing accounts.
- Do note that some fields cannot be changed, such as the **currency**of a **bank**or **cash account**.**Q3. What file formats are supported for importing COA?**

- Only the **.xlsx**(Excel Workbook) format using our template is supported for importing COA.

**Q4. Can I import COA from other accounting software I've used before?**

- You will need to transfer your COA information to our **Import COA**template and then import.

**Q5. How can I set a currency for an account while importing?**

- Only accounts with the account type **Asset - Bank Accounts**or **Asset - Cash**can have currencies set.
- Currencies can only be set for new accounts. Existing accounts cannot have their currencies changed after creation.
- To set a currency for new accounts while importing, select one of the eligible account types, and choose the currency that you want from the drop-down list in the template.
- **Note:**The currencies that show up in the drop-down list include your organization's base currency, as well as other currencies that you have added via Organization Details. For more information on adding currencies to your organization, refer to [Accounting](https://intercom.help/jaz-ca1acb882400/en/articles/9091111-accounting)

**Q6. Can I review the COA before importing?**

- Yes, you can.
- Before finishing your import, Jaz will let you review the list of accounts that will be created, updated, or currently have errors in the template.

**Q7. Will importing accounts affect any existing data in Jaz?**

- No, importing accounts will not affect any existing data.

**Q8. Are there any restrictions on the number of accounts I can import at once?**

- You can import up to a maximum of **1000 existing and new**accounts at the same time (In other words, 1000 rows of account data in the template.)

**Q9. What should I do if there are errors or duplicates in the imported COA? How can I fix them?**

- Jaz will highlight any issues with the rows/records that you have filled in within the import template.
- To fix it, go back to your account import template and amend the row with the error.
  - For example, if the error reads "Invalid account name" for Row 89, then check if the account on Row 89 has the account name filled in, or needs to be changed.
  - After fixing all the errors presented by Jaz, save your template and reimport the template to restart the bulk import process.

**Q10. Can I customize or modify the COA after import?**

- Yes, you can customize or modify the COA after import.
- After importing, you can find the imported accounts under **Settings > Configurations > Chart of Accounts.**
- Change the account fields for the account you would like to update.
- For more information about accounts in the COA, see [Chart of Accounts](https://help.jaz.ai/en/articles/8938028-chart-of-accounts-coa)

### Import Contacts
Source: https://help.jaz.ai/en/articles/9115782-import-contacts

**Q1. Can I import contacts in bulk?**

- Yes, you can import contacts in bulk using our **Import Contacts**template.
- The template can be found in the Contacts tab.
- Follow the steps on-screen to start importing your contacts.
- Review the imported list of contacts.
- You should see contacts that will be added, deleted, or with errors.
- If there are no issues, click on **Finish**to finish importing your contacts.
  - If not, continue adjusting and modifying your template, and click on **Back**to reupload.

**Q2. Can I update existing contacts using imports?**

- Yes, you can. When downloading the **Import Contacts**template, it will be pre-filled with contacts that you have already added in Jaz.
- You can modify the information in these rows relating to existing contacts.
- For example, you can change a contact from being just a **Supplier Contact**to a contact that is both a **Supplier**and a **Customer**.**Q3. What file formats are supported for importing contacts?**

- Only the **.xlsx**(Excel Workbook) format using our template is supported for importing contacts.

**Q4. Will importing contacts automatically create customer/supplier profiles?**

- Yes, importing contacts using our provided template will do this for you, provided that you have filled in the necessary details and also indicated the profile type (**Customer/Supplier/Both**) of the contact.**Q5. Can I import contacts from other accounting software I've used before?**

- You will need to transfer the contact information to our **Import Contacts**template, and import using our template.

**Q6. Will importing contacts affect any existing data in the accounting software?**

- No, the information in Jaz is self-contained and will not affect other software.

**Q7. Are there any restrictions on the number of contacts I can import at once?**

- You can import up to a maximum of **1000 existing and new**contacts at the same time (meaning 1000 rows of contact data in the template.)

**Q8. What should I do if there are errors or duplicates in the imported contacts? How can I fix them?**

- Jaz will highlight any issues with the rows/records that you have filled in within the import template.
- To fix it, go back to your contact import template and amend the row with the error.
- For example, if the error reads "Empty contact name" for Row 5, then check Contact Name is filled in for the contact on Row 5.
- After fixing all the errors presented by Jaz, save your template and reimport the template to restart the bulk import process.

**Q9. Can I review the list of imported contacts before importing?**

- Yes. Jaz will show a list of contacts that will be updated, created, deleted, or records having errors.

**Q10. Can I customize or modify the contact after importing contacts?**

- Yes, you can. After importing the contacts, find the contact that you would like to customize or modify by navigating to **Contacts.**
- For more information on managing contacts, you can refer to [Contacts.](https://help.jaz.ai/en/articles/8938009-contacts)

### Import Journals
Source: https://help.jaz.ai/en/articles/9115784-import-journals

**Q1. Can I bulk import multiple manual journals in bulk?**

- You can easily bulk import multiple manual journals from the journals tab.
- Follow the steps on the screen to start importing journal entries.
- Review the list of imported journal entries. You should see a list of journal entries that will be created or with errors.
- If there are no issues, click **Finish**to finish importing journal entries.
  - If not, continue adjusting and modifying your template, and click on **Back**to reupload.

**Q2. Can I update existing journals using import?**

- No, importing with the **Import Journal Entries**template will only create new journal entry lines.

**Q3. What file formats are supported for importing journal entries?**

- Only the **.xlsx**(Excel Workbook) format using our template is supported for importing COA.

**Q4. Can I import journals from an accounting software I have used before?**

- You will need to transfer the journal entries into our **Import Journal Entries**template and proceed using our template.

**Q5. Will importing the manual journal automatically update my financial records?**

- Yes, after importing the manual journal entries, you will see the new manual journal entries show up in the **Journals tab and financial statements. Q6. Will importing the manual journal affect any existing data in the accounting software?**

- No, the information in Jaz is self-contained and will not affect other software.

**Q7. Can I review the manual journal entries before finalizing the import?**

- Yes, you can. Before finishing your import, Jaz will let you review the list of journal entries that will be created or currently have errors in the template.

**Q8. What should I do if there are errors or discrepancies in the imported manual journal?**

- Jaz will highlight any issues with the rows/records that you have filled in within the import template.
- To fix it, go back to the **Import Journal Entries**template and amend the row with the error.
- For example, if the error reads "Missing Journal Reference" for Row 99, then check if the journal reference on Row 99 was entered.
- After fixing all the errors presented by Jaz, save your template and reimport the template to restart the bulk import process.
  - Alternatively, you can carry on with finishing the import process, and Jaz will skip over the rows with errors for you.

**Q9. Can I import a manual journal dated before the lock dates of my accounts?**

- No, any transactions including journal entries cannot be dated before the lock date of an account.
- This is because transactions for accounts cannot be modified from the day of the lock date and the period before it.
- As such, you will not be able to add new journal entries for these accounts.

**Q10. Are there any restrictions on the number of journals I can import at once?**

- You can import up to 1000 new journal entry lines at one time.

**Q11. Can I customize or modify the journals after import?**

- Yes, you can. After import, all the new journal entries will show up under **Journals.**
- Find the journal you would like to modify, and **Edit**to customize the journal entries as you need.

### Jaz Magic (Import attachments)
Source: https://help.jaz.ai/en/articles/9094452-jaz-magic-import-attachments

**Q1. What is Jaz Magic and how does it work?**

- Jaz Magic is a portfolio of magic features, providing an easy way for you to import existing invoices/bills into Jaz.
- You can simply drag & drop your invoice/bill documents or upload them using your native file manager.
- Jaz Magic will create a draft for you, and also try to autofill the invoice/bill details for you based on the uploaded attachments.

**Q2. What file types are supported by Jaz Magic?**

- Jaz Magic supports attachments with file types JPG, PNG, or PDFs.
- The file can only have a maximum size of 10MB.

**Q3. What information does Jaz Magic capture from the attachment?**

Jaz Magic will capture the following information:
- Invoice/Bill reference
- Due Date
- Invoice/Bill Date
- Items:
  - Item Name
  - Unit price
  - Unit
  - Discount
  - Quantity
  - Account
  - Tax Profile
- Contact Name

**Q4. Can Jaz Magic capture information from handwritten receipts?**

- Yes, as long as the information is readable Jaz Magic can read handwritten receipts as well.
- If the information is not readable, Jaz Magic will still create a draft invoice/bill with the handwritten receipt as an attachment.

**Q5. How do I know if Jaz Magic has successfully created the draft invoices or bills?**

- While the attachments are being uploaded, you should see a loading icon next to **Active Invoices**or **Active Bills.**
- If the attachments were successfully uploaded, you should see the icon turn green, and with the list of files uploaded.
- You can also check the drafts via the **Invoices > Draft** or**Bills > Draft **section.**Q6. How many drafts can be created at once by Jaz Magic?**

- You can drag and drop up to 50 attachments at once.

**Q7. Does Jaz Magic work with different languages?**

- No, Jaz Magic is currently only supported in English.

**Q8. Are the draft invoices/bills created by Jaz Magic any different from the draft invoices/bills created manually?**

- No, the draft invoices/bills function similarly. For more information about draft invoices and draft bills, you can refer to [Draft Invoices & Payments](https://help.jaz.ai/en/articles/9062346-draft-invoices-payments) or [Draft Bills & Payments](https://help.jaz.ai/en/articles/9104224-draft-bills-payments)

**Q9. Can I edit draft invoices/bills created by Jaz Magic?**

- Yes, you can edit draft invoices/bills created by Jaz Magic just like a normal draft invoice/bill.
- Refer to [Draft Invoices & Payments](https://help.jaz.ai/en/articles/9062346-draft-invoices-payments) or [Draft Bills & Payments](https://help.jaz.ai/en/articles/9104224-draft-bills-payments) for more information on editing draft invoices/bills.

**Q10. Can I convert drafts created by Jaz Magic to active?**

- Yes, you can convert draft invoices/bills created by Jaz Magic to active just like a normal draft invoice/bill.
- Refer to [How can I convert a draft bill to an active one?](https://help.jaz.ai/en/articles/9104224-draft-bills-payments) or [How can I convert a draft invoice to an active one?](https://help.jaz.ai/en/articles/9062346-draft-invoices-payments) for more information.

**Q11. Is Jaz Magic supported for supplier/customer credits?**

- Currently, Jaz Magic is not supported for supplier/customer credits.

**Q12. How does Jaz Magic extract contact information?**

Jaz Magic can extract contact details directly from uploaded attachments, creating or matching contacts automatically. Hereâ€™s how it works:
- **Upload an Attachment (e.g., Invoice, Bill)**
  - Upload the attachment that contains contact details (e.g., vendor or customer information).
- **Contact Extraction and Auto-Population**
  - Jaz Magic will detect contact information such as name, address, and VAT number. An option will appear to auto-populate the **Contact** field in the transaction form.
  - Selecting this option will automatically create a new contact entry based on the extracted details.
- **Matching with Existing Contacts**
  - If a matching contact is found, Jaz Magic links the extracted information to the existing contact to avoid duplicates.

**Q13. How does Jaz Magic handle updated item names?**

- Jaz Magic remembers updated item names for better classification. Future uploads with the same item and contact will use the updated name.
- If no match is found, the extracted name will be shown for editing.
