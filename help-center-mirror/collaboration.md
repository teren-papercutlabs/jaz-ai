### Collaboration
Source: https://help.jaz.ai/en/articles/9172516-collaboration

**Q1. What are the business transactions that I can create a comment thread in?**

- Currently, you can create a comment thread for:
  - Invoices
  - Bills

**Q2. How can I create a comment thread?**

- Threads can be created from the comment icon on the transaction details.

**Q3. How can I edit a comment?**

- Choose the edit option from the menu of the selected comment.
- Comments that have been edited will be shown with **(edited)** beside the comment.**Q4. What is reacting, and how does it work?**

- You can use emojis to react to your own, or another user's comment.

**Q5. Can I add my own emojis?**

- Currently, Jaz does not support adding custom emojis for comment threads.

**Q6. How does mentioning someone in a comment thread work?**

- Mentioning someone in a comment sends a notification to the user to view the comment thread.
- To mention someone, type **@**while writing a comment. A list of users in your organization will show up.
- After entering your comment, the user will be notified on the mobile app.
- You can also see comments where someone has mentioned you. Your name will show up in orange, as pictured below.

**Q7. If I mention someone while editing a comment, will the person be notified?**

- No, the person will not be notified after you have finished editing and saving the comment.

**Q8. What does it mean to resolve a comment thread?**

- Resolving a comment thread means that you and other users have finished settling any issues relating to that comment thread.

**Q9. How can I resolve a comment thread?Q10. Can I view comment threads that have already been resolved?**

- Yes, you can. Click on the **left and right chevron icons**to navigate between different comment threads that have been created for the transaction.
- The comment threads will include both active and resolved threads.

**Q11. How will I know if there is currently an active comment thread?**

- If there is a currently active comment thread on a transaction, the comment icon will have a notification badge on it.

**Q12. How will I know if there is activity on a comment thread that involves me?**

- You will receive a notification for activity in a comment thread that involves you, when:
  - Someone has mentioned you
  - New comments were added to a thread you previously commented on or were mentioned in
  - Someone reacts to your comment in the thread
- All notifications are received on the **Jaz Mobile App** for now.
  - Jaz is not only available on the web, but also on the [Google Play Store](https://jaz.ai/android) and the [iOS App Store](https://jaz.ai/ios)!
